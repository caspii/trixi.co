Background

This app is inspired by the likes of Kittysplit or Doodle which do not require you to register to do useful work (disclaimer: we actually made Kittysplit ðŸ˜‹)

There are already a huge number of todo-lists, project management tools, and so on, so why make another? Trixi is based on the assumption that simplicity wins. It has done away with all unnecessary features and left behind what we consider to be essential. If you think something essential is missing, please get in touch!

Trixi is ideal for small one-off events that you're organising with people you trust and like.

The easiest way to find out more it to give it a spin!

Technologies used

  • The webapp is a Python Flask application.
  • It runs on Google App Engine.
  • The frontend is styled with Bootstrap 4 using the Bootswatch Minty Theme.
  • All coding was done on Ubuntu using Atom.
