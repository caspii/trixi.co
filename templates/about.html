Background

Trixi is ideal for small one-off events that you're organising with people you trust and like.

This app is inspired by the likes of Kittysplit or Doodle which do not require you to register to do useful work (disclaimer: we actually made Kittysplit ðŸ˜‹)

There are already a huge number of todo-list apps, project management tools, etc. so why make another? Trixi is based on the assumption that simplicity wins. It has done away with all unnecessary features and what remains is only what we consider to be essential. If you think something essential is missing, please get in touch!

The easiest way to find out more it to give it a spin!

Created by Caspar.

Technologies used

  • This project is open source. Get the code on Github.
  • The webapp is a Python Flask application.
  • It runs on Google App Engine.
  • The frontend is styled with Bootstrap 4
  • All coding was done on Ubuntu using Atom.
